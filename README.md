# Baka.Pub Anycast DNS

The CoreDNS zones files deployed on anycast.baka.dn42.

Theoretically, it takes up to 5 minutes to deploy to each edge node.

## How to use?

Contact @bakabing via Telegram or send an email to bingxin@baka.pub to specify the required dn42 domain name. Then register your dn42 domain in the main dn42 git repository and submit a PR in this repository to deploy the zones files.
